Xmodeling Studio is an EMF plugin enabling the development of software applications by combining an executable model with standard methods written in Java. It allows adding method signatures to elements of an executable model and to execute them automatically during the interpretation of the model by the xDSL execution engine. It works for any xDSL defined with the EMF platform.
